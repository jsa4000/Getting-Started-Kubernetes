# Getting-Started-Kubernetes

## PlantUML

PlantUML is a component that allows to quickly write:

* Sequence diagram
* Usecase diagram
* Class diagram
* Object diagram
* Activity diagram (here is the legacy syntax)
* Component diagram
* Deployment diagram
* State diagram
* Timing diagram
